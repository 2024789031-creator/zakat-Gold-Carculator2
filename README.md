# Zakat Gold Calculator

A mobile application developed using Android Studio to calculate Zakat on gold based on Islamic guidelines.

## Features
- Calculate zakat based on gold weight and current gold price
- Supports two types of gold: Keep (Nisab: 85g) and Wear (Nisab: 200g)
- Real-time calculation as you type
- Input validation with error messages
- Share app link with others
- About page with developer information

## How to Calculate Zakat
1. Enter the gold weight in grams
2. Select the type of gold (Keep or Wear)
3. Enter the current gold value per gram (RM)
4. The app will automatically calculate and display the total zakat due

## Zakat Calculation Formula
- Total Gold Value = Gold Weight × Gold Value per gram
- Zakat Payable Value = (Gold Weight - Nisab) × Gold Value per gram
- Total Zakat Due = Zakat Payable Value × 2.5%
